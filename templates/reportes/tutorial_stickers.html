Tutorial de Stickers ZPL

Aprende a usar el sistema de generación de stickers ZPL para impresoras Zebra.

¿Qué son los stickers ZPL?

ZPL (Zebra Programming Language) es un lenguaje de programación utilizado por las impresoras Zebra para generar etiquetas y stickers de alta calidad.

Pasos para generar stickers:

  1. Configurar plantilla: Define el tamaño, campos y posición del código QR
  2. Seleccionar bienes: Elige los bienes patrimoniales para generar stickers
  3. Generar archivo ZPL: El sistema crea un archivo compatible con impresoras Zebra
  4. Imprimir: Envía el archivo a tu impresora Zebra

Impresoras compatibles:

  • Zebra ZD220
  • Zebra ZD410
  • Zebra ZD411
  • Otras impresoras compatibles con ZPL
